B) CSS Files and Table of contents - top

If you would like to edit the color, font, or style of any elements in one of these columns, go to folder css and open files custom.css. If you find that your new style is not overriding, it is most likely because of a specificity problem. Scroll down in your CSS file and make sure that there isn't a similar style that has more weight.
